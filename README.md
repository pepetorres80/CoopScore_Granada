# CoopScore Granada

Sistema inteligente de apoyo a la decisión para la identificación de oportunidades de desarrollo de vivienda cooperativa en la provincia de Granada.

## Descripción

CoopScore Granada es una plataforma desarrollada como Trabajo Fin de Máster en Data Science e Inteligencia Artificial.

La herramienta integra información demográfica, urbanística, económica y territorial para evaluar el potencial de los municipios de Granada mediante indicadores propios de viabilidad cooperativa.

## Funcionalidades

* Ranking municipal mediante CoopScore.
* Índice de Viabilidad Urbanística (IVU).
* Índice de Viabilidad Económica Cooperativa (IVEC).
* Mapa interactivo de oportunidades.
* Análisis individual de municipios.
* Consulta asistida mediante Inteligencia Artificial sobre documentación urbanística.

## Tecnologías utilizadas

* Python
* Streamlit
* Pandas
* Scikit-Learn
* SHAP
* Plotly
* Anthropic Claude

## Estructura metodológica

1. Integración y limpieza de datos.
2. Análisis exploratorio y reducción de dimensionalidad.
3. Construcción de indicadores multicriterio.
4. Validación mediante Explainable AI.
5. Desarrollo de dashboard interactivo.
